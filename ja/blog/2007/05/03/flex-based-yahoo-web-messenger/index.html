米 Y! のウェブメッセンジャーは Flex ベース

Tweet

米 Yahoo! が ブラウザ版Messengerをローンチ しました。業界動向的には MSN Web Messenger が既にありますし、目新しくもなんともないのですが、なんと 米Yahoo! 版は Flex ベースです。

証拠はこのキャプチャ。

コンボボックスのところが、いかにも Flex ですね。Flex を使ったことある人には、理解していただけるはず。

ついに、米 Y! というメジャーな Web 企業も Flex を一般向けサービスに導入したわけで、仕事で Flex を使ってみたい人は上司やお客さんの説得材料に使えるかもしれません。
