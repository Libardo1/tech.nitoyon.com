mixi で自分の都道府県を落とすアプリを作ってみた

Tweet

自分が住んでる都道府県をプロフィールページに貼っつけて、訪問した人に落としまくってもらうための mixi アプリを作ってみました。

サンプル

私の mixi アカウントに設置してます（足跡が気になる人は注意してね）。

  • http://platform001.mixi.jp/show_friend.pl?id=10037

インストールするには

※mixi アプリはβテスト中なのでちょっと手順が複雑になってます

  1. mixiアプリ オープンβ コミュニティーに参加する
  2. ○○県を落とすとどう跳ねるのか からアプリ登録する
  3. http://platform001.mixi.jp/home.pl を開くと自分のページにも追加されてる！

やってること

  1. OpenSocial の API でアプリを設置した人の住所を取得する
  2. 北海道を落とすとどう跳ねるのか？ の Flash を都道府県を渡しつつ埋め込む

とても簡単でした。隠すことは何もないのでソースコードも公開中。こちらからどうぞ。
