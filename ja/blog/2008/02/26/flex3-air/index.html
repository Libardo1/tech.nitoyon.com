Flex 3 SDK と AIR 1.0 公開されたけど…

Tweet

ついに、Flex 3 SDK と AIR 1.0 が公開されました。

  • akihiro kamijo: Adobe AIR 1.0 リリース

けど、Flex 2 Builder と Flex 2 SDK が落とせなくなった。困った。自分が探すのが下手なだけかと思って、うろうろしたけど見つからず。

米 Adobe の Flex ページ は完全に Flex Builder 3 と Flex 3 SDK に変わってる。http://www.adobe.com/go/flex_trial から More Flex downloads とか See all downloads をクリックしたが見つからず。Flex 2 の文字はなし。いろいろ探して、Adobe Flex Support Center に Flex 2 Builder への直リンクは見つけた。

日本 Adobe の Flex ページ には Flex 2 SDK とあるものの、リンク先は Flex 3 Builder に変わってる。Flex 3 SDK は日本語のサイトからは落とせない模様。

Adobe labs からは Hotfix3 つきの Flex 2.0.1 は落とせる模様。Hotfix2 はhttp://www.adobe.com/jp/support/flex/ts/documents/kb402000.htm から。

Flex 2 SDK/Builder をまだまだ現役で使ってる人も多いだろうに。大丈夫なのか。突然リンクが消えてしまう手法は Photoshop とか Illustrator のリリースでは許されるのかもしれないけど、開発ツールで同じことをやるのはやめてほしいものだ。
